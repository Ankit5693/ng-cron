QuartzCronComponent, UnixCronComponent Component

Selectors: quartz-cron, unix-cron
Inputs
  • cssClassPrefix
    The prefix will be used in css classes generating, for example you passed cssClassPrefix="my-" as a result the bootstrap class will be transformed from form-group to my-form-group.
    Type: string
    Default value: empty string ""
  • disabled
    The disabled state.
    Type: boolean
    Default value: false
  • localization
    Localization object
    Type: CronLocalization
  • tabs
    List of visible tabs.
    Type: Tab[]
  • hideTabs
    Control tabs visibility.
    Type: boolean
    Default value: false
  • activeTab
    The active tab.
    Type: Tab
Outputs
  • changed
    An event emitted when the user changed the cron value by UI.
    Event payload is the string of the newly cron value.
  • tabChanged
    An event emitted when the user changed the active tab by UI.
    Event payload is the new activated tab.
Methods
  • writeValue()
    Writes a new cron value to the component.
    writeValue(cronValue: string): void
    Parameter cronValue: string
    Returns: void
